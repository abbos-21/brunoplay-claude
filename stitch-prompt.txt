Build a complete mobile-only Telegram Mini App game UI called "Bruno's Fishing Adventure". This is a tap-to-earn fishing game with mini-games, card upgrades, and crypto withdrawal. The design should feel premium, playful, and game-like — think Hamster Kombat or Notcoin but with a fishing/ocean theme. Use a dark navy-to-deep-blue gradient background throughout, with golden/amber accents for coins, cyan/electric-blue for stars (premium currency), and soft white text. All screens are mobile-only (375px width, full viewport height). Use rounded cards, subtle glassmorphism, and smooth micro-interactions.

=== COLOR PALETTE ===
- Background: dark navy gradient (#0a1628 → #132042)
- Cards/Surfaces: semi-transparent dark (#1a2744 with 80% opacity, glass blur)
- Primary accent: golden amber (#f5a623) for coins and CTAs
- Secondary accent: cyan blue (#00d4ff) for stars and premium elements
- Success: emerald (#10b981)
- Danger: coral red (#ef4444)
- Text primary: white (#ffffff)
- Text secondary: muted blue-gray (#8b9dc3)
- Tab bar active: golden amber, inactive: muted gray

=== GLOBAL LAYOUT ===
Every screen shares:
- Top bar: user level badge (left), coin counter with coin icon (center-left), star counter with star icon (center-right), user avatar circle (right)
- Bottom tab bar with 5 tabs: Home (fish icon), Games (gamepad icon), Tasks (checklist icon), Friends (users icon), Wallet (wallet icon)
- The currently active tab is highlighted with the golden accent color
- Safe area padding for iOS notch and bottom bar

=== DESIGN PRINCIPLES ===
- Every interactive element should have visible touch feedback
- Use skeleton loading states for data-heavy screens
- Coin and star amounts should use locale-formatted numbers (1,234,567)
- All modals slide up from bottom (bottom sheet style)
- Success states use confetti/particle animations
- Errors shown as red toast notifications at top
- Transitions between screens use horizontal slide animations
- The overall feel should be premium mobile gaming — not a web app
- Support both English and Russian languages (RTL not needed, just text swapping)
- All icons should use a consistent outline/filled style (Lucide or similar)
- Glassmorphism cards: semi-transparent backgrounds with subtle blur
- Rounded corners everywhere (16px for cards, 12px for buttons, 999px for pills)
- Font: Inter or system font, bold headings, medium body text
